# Zuma Unity Project (Diploma Thesis) 🐸🪐

This project is a modern reconstruction of the classic Zuma-style marble shooter, developed as a diploma thesis for the final year of study. The game is built using **Unity 6** and **C#**.

## 🎮 Game Overview
The player controls a central turret (the frog) that fires colored orbs. The objective is to eliminate a moving chain of marbles by creating matches of three or more identical colors before they reach the "skull" at the end of the path.

## 🚀 Key Technical Features
* **Engine:** Unity 6 (6000.0.x).
* **Design Patterns:** Implemented the **Singleton Pattern** for robust Sound and Scene management.
* **Dynamic Audio System:** Features automatic background music transitions between the Menu and Gameplay scenes, and randomized SFX for object destruction.
* **Scalable UI:** A fully responsive User Interface including a Main Menu, Pause functionality, and a dynamic "About" section linking to this repository.
* **Cross-Platform Build:** Optimized Windows executable with handled Aspect Ratio adjustments.

## 📦 How to Run
1. Download the latest release or the `Build_Zuma` folder.
2. Run `Zuma Game Project.exe`.
3. Use the mouse to aim/shoot and **ESC** to pause the game.

## 📂 Repository Structure
* `/Assets` — Contains all C# scripts, Prefabs, Audio clips, and Sprites.
* `/ProjectSettings` — Internal Unity configuration files.
* `.gitignore` — Optimized for Unity projects to ensure a clean repository.
